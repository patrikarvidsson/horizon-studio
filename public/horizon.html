Projects / Horizon

A personal wiki

Horizon is the codebase running the website you are currently browsing.

The purpose of horizon is to keep track of the time spent researching different topics and how much time is invested on each individual project.

It summarises logged data and provides key insights into productivity and balance across different sectors.

Data

Horizon parses ndtl/tbtl human-readable flat-file databases for all wiki entries and recorded logs.

Data is summarized and processed to visualize productivity on each project. Some of this data can be seen at the statistics page.

Design

Horizon is an experiement in trying to find a balance between minimalism and good user experience. It tries to promote readability for the repository of information it hosts.

Building

The current iteration of horizon is built with Node and its pages are generated with javascript. A lot of the codebase is inspired by ara, a wiki project created by Josh Avanier.

The HTML and CSS are kept to a minimum to ensure a simple and fast website. The pages are hosted at Vercel.

Further reading

For more information on the concepts of personal assistants and time management, visit XXIIVV or Victor Ivanov.
